# 三层架构（GameEngine + RunState + CombatState）实现计划

## 背景

将现有的单层 GameState 拆分为三层架构：

* **GameEngine**：顶层入口（**非单例！可多实例并行**）

* **RunState**：持久层，伴随玩家从 Act 1 到通关心脏

* **CombatState**：易失层，仅存在于一场战斗

***

## ⚠️ 重要修正（三大地雷）

### 地雷1：GameEngine 绝对不能用单例模式！

**正确方案**：

```cpp
// GameEngine 是普通类，可被多次实例化
class GameEngine {
    std::shared_ptr<RunState> runState;
    std::shared_ptr<CombatState> combatState;
    ActionManager actionManager;
    EventBus eventBus;
};

// 人类游玩：main() 里 new 一个
auto engine = std::make_unique<GameEngine>();

// AI 训练：for 循环里 new 50 个完全独立的对象
std::vector<std::unique_ptr<GameEngine>> engines;
for (int i = 0; i < 50; ++i) {
    engines.push_back(std::make_unique<GameEngine>());
}
```

## ⚠️ 重要落地细节

### 落地细节1：CRTP 模板必须加 `using` 声明！

**错误写法（编译报错）**：

```cpp
template <typename Derived>
class CloneableCard : public AbstractCard {
public:
    std::shared_ptr<AbstractCard> clone() const override {
        return std::make_shared<Derived>(static_cast<const Derived&>(*this));
    }
};

class StrikeCard : public CloneableCard<StrikeCard> {
public:
    StrikeCard() : CloneableCard("Strike", 1, CardType::ATTACK, CardTarget::ENEMY) {} // ❌ 报错！
};
```

**正确写法（必须加 using）**：

```cpp
template <typename Derived>
class CloneableCard : public AbstractCard {
public:
    // 🌟 极其关键的一行：把 AbstractCard 的构造函数直接"吸"上来
    using AbstractCard::AbstractCard;

    std::shared_ptr<AbstractCard> clone() const override {
        return std::make_shared<Derived>(static_cast<const Derived&>(*this));
    }
};

// 现在子类可以完美调用父类的构造参数，一点错都不会报！
class StrikeCard : public CloneableCard<StrikeCard> {
public:
    StrikeCard() : CloneableCard("Strike", 1, CardType::ATTACK, CardTarget::ENEMY) {} // ✅ 正常
};
```

***

### 落地细节2：AbstractAction::update() 参数改为 GameEngine&

**问题**：Action 需要同时访问 CombatState（战斗临时数据）和 RunState（永久数据）。

**正确方案**：

```cpp
class AbstractAction {
public:
    virtual ~AbstractAction() = default;

    // 🌟 改为接收 GameEngine&，可访问完整上下文
    virtual bool update(GameEngine& engine) = 0;
};

// 示例：GainGoldAction 修改 RunState
bool GainGoldAction::update(GameEngine& engine) {
    engine.runState->gold += this->amount;  // 改永久数据
    return true;
}

// 示例：DamageAction 修改 CombatState
bool DamageAction::update(GameEngine& engine) {
    engine.combatState->monsters[0]->takeDamage(this->damage);  // 改临时数据
    return true;
}
```

### 第一步：创建 GameEngine 类（非单例）

**目标**：创建可多实例化的 GameEngine。

**实现内容**：

```
src/engine/
├── GameEngine.h      # 类声明
└── GameEngine.cpp    # 类定义
```

**GameEngine 结构**：

```cpp
class GameEngine {
public:
    GameEngine();  // 默认构造
    ~GameEngine();

    // 禁用拷贝，支持移动
    GameEngine(const GameEngine&) = delete;
    GameEngine& operator=(const GameEngine&) = delete;
    GameEngine(GameEngine&&) = default;
    GameEngine& operator=(GameEngine&&) = default;

    // 持久层（一局游戏）
    std::shared_ptr<RunState> runState;

    // 易失层（战斗中，nullptr = 不在战斗）
    std::shared_ptr<CombatState> combatState;

    // 全局动作管理器
    ActionManager actionManager;

    // 全局事件总线
    EventBus eventBus;

    // ========== 核心接口 ==========

    // 开始新游戏
    void startNewRun(unsigned int seed);

    // 进入战斗
    void startCombat(std::shared_ptr<Monster> monster);

    // 结束战斗
    void endCombat();

    // 主循环推进
    void tick();
};
```

**CombatState 结构**：

```cpp
class CombatState {
public:
    // 玩家引用（指向 RunState.player，不拷贝！）
    std::shared_ptr<Player> player;

    // 怪物
    std::vector<std::shared_ptr<Monster>> monsters;

    // 5个物理牌堆（从 masterDeck 深拷贝）
    std::vector<std::shared_ptr<AbstractCard>> drawPile;
    std::vector<std::shared_ptr<AbstractCard>> hand;
    std::vector<std::shared_ptr<AbstractCard>> discardPile;
    std::vector<std::shared_ptr<AbstractCard>> exhaustPile;
    std::vector<std::shared_ptr<AbstractCard>> limbo;

    // 战斗内临时获得的卡牌（战斗结束后清空）
    std::vector<std::shared_ptr<AbstractCard>> temporaryCards;

    // 战斗 RNG（与 RunState.rng 隔离）
    RandomManager combatRng;

    // 状态
    bool isPlayerDead = false;
    bool isMonsterDead = false;
    StatePhase currentPhase = StatePhase::BATTLE_START;

    // ========== 核心接口 ==========

    // 工厂方法：从 RunState 创建 CombatState
    static std::unique_ptr<CombatState> createFromRun(
        RunState& runState,
        std::shared_ptr<Monster> monster,
        unsigned int seed);

    // 战斗结束：清理临时状态
    void cleanup();

private:
    // 深拷贝卡牌（关键！）
    static std::vector<std::shared_ptr<AbstractCard>> deepCopyDeck(
        const std::vector<std::shared_ptr<AbstractCard>>& masterDeck);
};
```

**战斗初始化流程**：

```cpp
std::unique_ptr<CombatState> CombatState::createFromRun(
    RunState& runState,
    std::shared_ptr<Monster> monster,
    unsigned int seed) {

    auto combat = std::make_unique<CombatState>();
    combat->combatRng = RandomManager(seed);

    // 1. 玩家引用（不拷贝！）
    combat->player = runState.player;

    // 2. 怪物
    combat->monsters.push_back(monster);

    // 3. 深拷贝 masterDeck → drawPile（关键！）
    combat->drawPile = deepCopyDeck(runState.masterDeck);
    shuffleDeck(combat->drawPile, combat->combatRng);

    // 4. 抽初始手牌
    drawInitialCards(*combat, 5);

    return combat;
}
```
