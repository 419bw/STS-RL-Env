# Complexity Criteria (Decision Matrix)

## 🟢 Simple Task (Direct Dispatch)
- **特征**: 
  - 单一函数修改或添加。
  - 纯文本生成、翻译、格式化。
  - 不涉及核心架构变动的配置调整。
  - 明确的、无歧义的指令。
- **动作**: 直接调用最匹配的单个 Agent，无需流水线。

## 🔴 Complex Task (Full Pipeline Trigger)
- **特征**: 
  - **架构级变动**: 新增类、修改接口定义、涉及多模块交互。
  - **安全敏感**: 涉及用户输入处理、权限控制、RL 奖励机制、网络通信。
  - **逻辑复杂**: 状态机变更、并发处理、死锁风险高。
  - **模糊需求**: 需要拆解、方案设计才能实施的任务。
- **动作**: 必须启动 **Phase 1 -> Phase 4** 完整流水线。
